# 编程实践

## Typescript

常用类型的命名方式

```typescript
import { type Object as ObjectUtil } from "ts-toolbelt";

// 查询接口
interface StudentDto {
  id: number;
  name: string;
  age: number;
}

// 更新接口
interface StudentUpdateDto
  extends ObjectUtil.Optional<StudentDto, "name" | "age"> {}

// 新增接口
interface StudentCreateDto extends Omit<StudentDto, "id"> {}
```
